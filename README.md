# Welcome to TapperTelemetry

TapperTelemetry is a powerful tool for monitoring and analyzing gameplay data.

## Installation Guide

### BepInEx Installation
1. Download the latest version of BepInEx from the [BepInEx GitHub releases page](https://github.com/BepInEx/BepInEx/releases).
2. Extract the contents of the downloaded zip file into the root folder of your game.
3. Launch the game; BepInEx will create a folder named `BepInEx` in your game directory, which indicates that the installation was successful.
4. Place the `TapperTelemetry.dll` file in the `BepInEx/plugins` directory.
