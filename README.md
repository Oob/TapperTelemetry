# Welcome to TapperTelemetry

Welcome to Tapper Telemetry! This unity plugin is designed to help users seamlessly monitor and manage their telemetry data for optimal 
performance and analysis. It is mostly a bepinex plugin for monitoring and analyzing gameplay data, with a included viewer for reviewing live 
and exported telemetry data files.

## Installation Guide

### BepInEx Installation
1. Download the latest version of BepInEx from the [BepInEx GitHub releases page](https://github.com/BepInEx/BepInEx/releases).
2. Extract the contents of the downloaded zip file into the root folder of your game.
3. Launch the game; BepInEx will create a folder named `BepInEx` in your game directory, which indicates that the installation was successful.
4. Place the `TapperTelemetry.dll` file in the `BepInEx/plugins` directory.
5. Expect it to break, this has only been tested with one proof of concept.

# Tapper Telemetry
## Features
- **Real-time Monitoring**: Track telemetry data as it comes in, ensuring you always have the most up-to-date information.
- **User-Friendly Interface**: Navigate through features effortlessly with our intuitive layout.
- **Customizable Dashboards**: Set up your dashboards to display the data that matters most to you.

## Installation
To install Tapper Telemetry, please follow these steps:
1. Download the latest version from our [releases page](https://github.com/Oob/TapperTelemetry/releases).
2. Extract the downloaded folder to your game directory's bepinex plugin folder. Commonly found at
`C:\Program Files (x86)\Steam\steamapps\common\<game>\BepInEx\plugins\`. Extract it as a `\TapTelemetry` folder to the `\plugins` directory.

## Usage
Once installed, you can start using Tapper Telemetry by launching the application. It will automatically launch the included html based viewer
with your game session.

Here are some quick tips to get started:
- Explore the dashboard to view real-time telemetry data.
- Options are saved to your browser's local storage.
- Everything is experimental, please create an issue if you find a problem.
